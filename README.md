#  Gestor de Torneos de Fútbol

App para la administración y organización de torneos de fútbol.

Funcionalidades:

- Registro e inicio de sesión de usuarios
- Creación y administración de torneos
- Gestión de equipos
- Gestión de jugadores
- Generación de fixture
- Carga de resultados
- Tabla de posiciones
- Tabla de goleadores
- Estadísticas del torneo

Tecnologías:

Backend
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- JWT Authentication

Frontend
- Angular

Estado del proyecto

 En desarrollo

Objetivo:

Desarrollar una plataforma que permita gestionar torneos de fútbol de manera simple, centralizando equipos, jugadores, partidos y estadísticas en una única aplicación.
